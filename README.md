# Google Pay Expense Sharing Application

## Overview

This project is a Python-based Expense Sharing Application inspired by Google Pay's expense-sharing feature. It helps users split expenses among friends, track transactions, calculate settlements, and suggest payments required to settle balances.

## Features

* Add expenses among multiple users
* Equal expense splitting
* Transaction history tracking
* Settlement calculation
* Payment suggestions
* Expense matrix visualization

## Technologies Used

* Python
* NumPy
* PrettyTable

## Installation

Install required packages:

pip install numpy prettytable

## Running the Project

Open the Jupyter Notebook:

expense_sharing.ipynb

Run all cells and follow the menu options.
